Enterprise Safety

5-layer guardrail system

Because AI + production infrastructure requires real protection.

Layer 1

Operation Classification

Every tool classified as READ · CONFIG · WRITE · DESTRUCTIVE before execution.

Layer 2

Safety Mode

read_only (default) → standard → unrestricted. Start safe, escalate with intention.

Layer 3

Permanent Denylist

74 operations that can never run regardless of safety mode. delete_cluster, drain_node, and more.

Layer 4

Confirmation Gate

All write operations require explicit confirmation with a unique ID before executing.

Layer 5

Audit Log

Every operation logged with timestamp, user, safety mode, and outcome. SOC2 ready.

One-line install

Up in 60 seconds

No config files. No YAML. Just install and ask.

Claude Code / Terminal
# Step 1 — Install
pip install eks-sage-enterprise

# Step 2 — Register with your MCP client
claude mcp add eks-sage -s user -- python -m eks_sage_enterprise.server

# Step 3 — Connect and ask
connect to cluster my-cluster in us-east-1
give me a full cluster health check
